# Caesar Cipher Python Tool

A simple, lightweight Python script that allows users to encrypt and decrypt text using the classic **Caesar Cipher** algorithm.

## 📖 Description
The Caesar Cipher is one of the simplest and most widely known encryption techniques. It is a type of substitution cipher in which each letter in the plaintext is replaced by a letter some fixed number of positions down the alphabet. 

For example, with a left shift of 3, `D` would be replaced by `A`, `E` would become `B`, and so on.

## 🚀 Features
- **Encrypt and Decrypt:** Switch between modes easily.
- **Customizable Shift:** Use any integer as the shift key.
- **Case Sensitivity:** Maintains the original capitalization of the text.
- **Symbol Preservation:** Spaces, numbers, and punctuation remain unchanged.
